üõ†Ô∏è Step 2: Check Disk Partition Style (Windows & Linux)

üíª On Windows:

  • Press Win + X ‚Üí Open Disk Management

üêß On Linux:

üîπ gpt ‚Üí use UEFI boot
üîπ msdos ‚Üí use Legacy boot

üõ†Ô∏è Fix Proxmox Repo Error After Fresh Installation

After running apt update right after installing Proxmox VE, you might see this error:

ü§î Why This Happens

Proxmox by default enables the Enterprise Repository, which is meant for users with a paid subscription. If you‚Äôre using Proxmox for personal or homelab use (without a subscription), you‚Äôll need to switch to the No-Subscription Repository.

‚úÖ Fix: Use No-Subscription Repository

üíæ Reclaim Full Internal SSD for VM Storage

üîå Connect & Configure External SSD for VM Storage

Why I did this:

To save space on my internal SSD and keep things organized. Instead of putting VMs, backups, and ISOs on the main drive, i moved them to an external SSD.

‚úÖ Keeps the system clean
‚úÖ Gives more storage for VMs
‚úÖ Easier to manage everything

Optional: LVM-Thin on External SSD

It helps you save space and create copies of VMs faster.

Normally, if you make 3 copies of a 20GB VM, it uses 60GB. With LVM-Thin, the copies share space ‚Äî like magic ‚ú® ‚Äî so it uses much less.

‚úÖ Good if you make/test lots of VMs
‚úÖ Makes cloning and snapshots faster and lighter

In short: You‚Äôd set this up if you want to do more with less space. If not doing that, you can skip it üôÇ
